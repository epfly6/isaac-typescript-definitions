PickEnemyTarget

  • PickEnemyTarget(maxDistance: number, frameInterval?: number, flags?: number, coneDir?: Vector, coneAngle?: number): void
    • Defined in EntityFamiliar.d.ts:32

    Parameters

    • maxDistance: number
    • Optional frameInterval: number

      Default is 13.

    • Optional flags: number

      Default is 0. A combination of the following flags (none of these are set by default): 1: Allow switching to a better target even if we already have one 2: Don't prioritize enemies that are close to our owner 4: Prioritize enemies with higher HP 8: Prioritize enemies with lower HP 16: Give lower priority to our current target (this makes us more likely to switch between targets)

    • Optional coneDir: Vector

      Default is Vector.Zero. If ~= Vector.Zero, searches for targets in a cone pointing in this direction.

    • Optional coneAngle: number

      Default is 15. If ConeDir ~= Vector.Zero, sets the half angle of the search cone in degrees (45 results in a search angle of 90 degrees).

    Returns void
